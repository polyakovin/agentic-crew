# Hermes Package: Agent Tester

Portable Hermes-style package for the Agent Tester specialist.

This package adapts the Agentic Crew harness in `agents/agent-tester/` to Hermes
usage. It does not introduce a new protocol; Hermes-facing tasks should still
emit Agentic Crew-compatible payloads.

## Files

- `manifest.yaml`: Hermes package metadata.
- `instructions.md`: Hermes-facing operating instructions.

Status: draft-ready pending validation against the concrete Hermes runtime.
