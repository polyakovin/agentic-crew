# Protocol Steward Harness

Copy this folder when a project needs harness, role-card, Agent Card, pack, or A2A
profile review as a specialist A2A agent.

- `role.md`: protocol stewardship instructions.
- `agent-card.json`: A2A discovery card.
- `harness.yaml`: local wiring, accepted payloads, emitted payloads, and gates.
- `operations.md`: source map, workflow, tool policy, rubric, eval seeds, and
  release notes.
- `run-record.template.json`: harness/protocol review record template.

Use this specialist before adding broad new agents or changing the shared
interaction profile.
