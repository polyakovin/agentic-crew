# Devlog / Community Marketing Specialist Evaluation Plan

## Seed Cases

### DM-001: Devlog Post from Git Log
**Input:** Git log with 12 commits (8 gameplay, 4 technical).
**Expected:** 1-3 paragraph devlog post, gameplay changes only, every claim backed by spec references.
**Gate:** Gameplay-first, no technical jargon, matches devlog/template.md.

### DM-002: Release Notes from Feature Diff
**Input:** Spec diff for new Endless mode feature set.
**Expected:** Condensed player-facing release notes: what changed → what it gives.
**Gate:** No "we refactored X" or variable names.

### DM-003: Itch.io Page Refresh
**Input:** Current itch.io page copy + current docs/game.md.
**Expected:** Updated description reflecting current feature set, correct tags, status migration wording.
**Gate:** Game = Locksmith (not Safe Cracker), legacy URL as migration.

### DM-004: Social Copy from Screenshot
**Input:** Screenshot + feature description.
**Expected:** 1-2 line social post + caption for progress image.
**Gate:** Player-facing benefit, no implementation details.

### DM-005: Claim Verification — Implemented
**Input:** Spec reference + make check evidence for crank feel improvement.
**Expected:** Devlog paragraph about crank improvement, citing make check pass.
**Gate:** Claim backed by spec + build.

### DM-006: Claim Verification — Not Implemented
**Input:** Spec draft for unreleased safe dial feature.
**Expected:** Agent refuses to write about it, flags as unverifiable.
**Gate:** No unverified claims in output.

### DM-007: Tonally Correct In-Game Handoff
**Input:** User asks to write loreLine for new item.
**Expected:** Handoff to Narrative Noir Copy Specialist with context.
**Gate:** No in-game copy written directly.

### DM-008: Screenshot Asset Handoff
**Input:** Devlog post needs screenshot of new UI.
**Expected:** Visual asset need documented for Visual Art Specialist.
**Gate:** Concrete screenshot reference, not improvised placeholder.

### DM-009: Legacy URL Handling
**Input:** Itch.io page refresh task mentioning safe-cracker URL.
**Expected:** Safe Cracker referenced as "legacy URL / status migration", not "also known as".
**Gate:** Game = Locksmith, URL = legacy polyakovin.itch.io/safe-cracker.

### DM-010: Build Evidence Inclusion
**Input:** Devlog post about adventure mode phases.
**Expected:** make check output included or honestly reported as missing.
**Gate:** Evidence or honest gap report.
